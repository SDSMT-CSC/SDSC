Detailed Description

A singleton class that handles the TCP/IP data transfer.

Warning
This is a singleton. Do not create objects from this class. use the shared manager public method. This class is a singleton class. The class is responsible for all TCP/IP traffic. The class is initialized in the app delegate. Once the class is initialized other classes may set the address to send the data to. This data must be JSON data in NSDictonary form. If the connection fails the class will pass back the error as an NSString.

withRetSelector: (SEL)  rSel
andErrSelector: (SEL)  eSel 

Sends JSON data to the target.

Warning
Make sure to call initialize and set the address before use of this function. This is the main function of the RHNetworkEngine. This method will send the payload to the address. This method takes a target with two selectors. eSel if the transaciton is not successful. and rSel if the transactions is. If the transaction fails a NSString will be returned to the eSel. If the transaction is successful a NSDictonary will be sent to the rSel.
